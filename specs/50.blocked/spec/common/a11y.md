## 역할
비네이티브 키보드 활성 불가 요소 (`<span>` / `<div>` / `<img>` 등) 에 `onClick` 을 바인딩할 때, **마우스/터치 사용자와 키보드/스크린리더 사용자에게 동일한 활성 경로를 제공**하기 위한 공용 a11y 계약. 본 spec 은 헬퍼 (`activateOnKey`) 자체의 계약, 호출부 패턴 B 의 시스템 불변식, 그리고 재발 방지 grep 게이트를 박제한다. 본 spec 은 패턴 B "공용 접근성 불변식" 한정 — 포커스 링/대비/aria-label 텍스트 품질/스크린리더 낭독 텍스트는 별도 a11y 축.

> **용어 충돌 주의**: 본 문서의 "패턴 B" 는 **a11y 패턴 B** (`role="button"` + `tabIndex={0}` + `onKeyDown={activateOnKey(...)}`) 를 가리킨다. `30.spec/blue/common/test-idioms.md` 의 "패턴 A / 패턴 B" 는 MSW 수명주기 패턴 — 동음이의이며 무관.

## 공개 인터페이스

### `activateOnKey(handler)` (`src/common/a11y.js`)
- 시그니처: `(handler: (event: KeyboardEvent) => void) => (event: KeyboardEvent) => void`.
- 동작:
  - `event.key === 'Enter' || event.key === ' '` 일 때만 `event.preventDefault()` 호출 후 `handler(event)` 위임.
  - 그 외 키는 무시 (이벤트 전파 자체는 차단하지 않음).
- 사용 금지 영역: `<input>` / `<textarea>` 위 (Space/Enter 가 이미 텍스트 입력·폼 제출 의미를 가짐). 호출부 책임으로 문서화.

## 패턴 B 불변식 (REQ-20260421-033 FR-01)

런타임 소스 (`src/**/*.{jsx,tsx}`, `*.test.*` 제외) 에서 다음 조건이 동시 성립한다:

1. **포지티브 — 동시 부여**: `<span>` / `<div>` / `<img>` / `<li>` / `<p>` 등 네이티브 키보드 활성 불가 요소에 `onClick` 핸들러를 바인딩한 모든 지점은 동일 요소에 다음 4 속성을 동시에 부여한다.
   - `role="button"` (또는 의미상 등가 ARIA role: `link`, `tab`, `menuitem` 중 호출부 의미에 맞는 단일 선택)
   - `tabIndex={0}`
   - `onClick={<handler>}`
   - `onKeyDown={activateOnKey(<동일 handler 참조>)}`

2. **핸들러 단일 참조**: `onClick` 와 `onKeyDown` 이 동일 함수 참조를 공유한다. arrow function 직접 선언이 필요하면 로컬 변수 (`const handle = ...`) 로 추출 후 두 핸들러에 동일 변수 참조를 주입한다 — `onClick={() => fn()}` + `onKeyDown={activateOnKey(() => fn())}` 형태로 두 arrow 가 별개 인스턴스로 생성되는 패턴은 의미상 등가하지만 회귀 검증 가독성 저하로 권장하지 않는다.

3. **예외 (Negative — 패턴 B 부여 면제)**: 다음 요소·구성은 본 불변식의 적용 대상에서 제외하며, 본 spec §예외 절에 명시적으로 열거된 경우에만 면제된다.
   - 네이티브 활성 요소: `<button>`, `<a href>`, `<input>`, `<textarea>`, `<select>`, `<summary>` 등 (브라우저가 키보드 활성 기본 제공).
   - 네이티브 활성 요소를 wrapping 하는 비활성 마커: react-router `<Link>` 의 자식 `<span>` 처럼, 부모 요소가 이미 포커스 대상이고 Enter 활성을 처리하는 경우 (§예외 §M5 참조).
   - 의미상 비활성 표시 요소: `role="listitem"` 등 활성 의미를 시사하지 않는 ARIA role 만 사용하고 `onClick` 도 없는 요소.

> **role ↔ behavior 일관성 원칙 (WAI-ARIA 1.2 §5.2.5)**: 요소가 `role="button"` 을 선언하면 키보드 사용자에게도 button 으로 활성 가능해야 한다. `role="button"` + `onClick` + `tabIndex` 누락 / `onKeyDown` 누락 은 ARIA 사용 안티패턴.

## 예외 (Negative 열거)

본 불변식의 면제 지점은 명시적으로 열거하며, 면제 사유와 회귀 점검 단서를 함께 박제한다.

### Native 활성 요소 (자동 면제)
- `<button data-testid="newlog-button" role="button">` (`src/Log/Log.jsx:21`) — 네이티브 `<button>` 이므로 패턴 B 의 `tabIndex` / `onKeyDown` 부여 불필요. `role="button"` 은 명시적 ARIA 라벨 보강 목적.

### react-router `<Link>` 내부 비활성 마커 (M5 — REQ-033 FR-05 **확정 면제** — TSK-20260421-77 @3971a46)

- **회귀 baseline (블록 경계 준수)** — 본 §예외 불변식은 DOM 어설션 레이어를 **우선 gate** 로 채택하며 grep 레이어는 **보조 gate** 로 블록 경계 준수 idiom 을 채택한다 (REQ-20260421-040 FR-04). 보조 gate: `sed -n '117,123p' src/Log/LogItemInfo.jsx | grep -E '(tabIndex|onKeyDown)'` → **0 match** (edit-button opening tag L117-123 범위 내 `tabIndex`/`onKeyDown` 속성 부재). 인접 L126-134 `<span role="button" data-testid="delete-button" tabIndex={0} onKeyDown={activateOnKey(handleDelete)}>` 의 속성은 본 gate 범위 외. **사용 금지 패턴 (false-positive)**: `rg -nU --multiline 'data-testid="edit-button"[\s\S]{0,N}?(tabIndex|onKeyDown)'` — `[\s\S]{0,N}` 이 JSX 태그 경계 (`</span>`, `</Link>`) 를 존중하지 않아 인접 `delete-button` 의 `tabIndex` 까지 도달 → **1 match false-positive** (§스코프 규칙 (c) 참조).

  - **경고**: `rg -nU --multiline` 의 `[\s\S]*?` / `[\s\S]{0,N}` 패턴은 JSX 태그 경계 (`<tag …>` opening / `</tag>` closing) 를 **존중하지 않는다**. 인접한 형제 요소 (예: `delete-button` 이 `edit-button` 직후에 나오는 경우) 의 속성까지 매치되어 **블록 내부에는 부여되지 않은 `tabIndex`/`onKeyDown` 을 거짓 양성으로 보고** 한다.   - **권장 idiom (택1)**:
    1. **고정 라인 범위** (가장 단순·강건) — JSX opening tag 의 시작/종료 라인을 스냅샷으로 박제 후 `sed -n '<start>,<end>p' <file> | grep -E '<attr-pattern>'`. 예: M5 edit-button 은 `sed -n '117,123p' src/Log/LogItemInfo.jsx | grep -E '(tabIndex|onKeyDown)'` → 0 match (HEAD=d9d026b 실측). JSX 편집 시 라인 번호 유지 책임은 회귀 테스트로 보완.
    2. **DOM 어설션 우선 gate** — grep 보조·테스트 레이어 주결. `el.getAttribute('tabIndex') === null` / `onkeydown === null` 형태로 런타임 트리 속성 부재 확인. M5 테스트 레이어 (`src/Log/LogItemInfo.test.jsx:270 describe(... M5 Edit Link span §예외)`) 가 2 it 로 박제.
